## 三、ESP32 ↔ TB6612FNG 接線

### TB6612FNG 接腳說明
```
         TB6612FNG 模組
  ┌────────────────────────────────┐
  │  PWMA ◄── ESP32 GPIO25 (PWM)  │  AO1 ──► 左馬達 (+)
  │  AIN1 ◄── ESP32 GPIO26        │  AO2 ──► 左馬達 (-)
  │  AIN2 ◄── ESP32 GPIO27        │
  │  PWMB ◄── ESP32 GPIO32 (PWM)  │  BO1 ──► 右馬達 (+)
  │  BIN1 ◄── ESP32 GPIO33        │  BO2 ──► 右馬達 (-)
  │  BIN2 ◄── ESP32 GPIO14        │
  │  STBY ◄── 3.3V (永遠致能)     │
  │  VM   ◄── 7.4V (馬達電源直供) │
  │  VCC  ◄── 3.3V (邏輯電源)     │
  │  GND  ◄── 共同 GND            │
  └────────────────────────────────┘

  ⚠️  TB6612FNG 是 3.3V 邏輯，ESP32 直接接！不需要電平轉換。
  ⚠️  L298N 的 ENA/ENB 跳線在 TB6612FNG 上沒有，STBY 接 3V3 即可。
```

### 方向控制真值表
```
  AIN1  AIN2  PWMA  → 馬達A動作
   H     L    PWM   → 正轉 (前進)
   L     H    PWM   → 反轉 (後退)
   L     L     X    → 短路煞車 (快速停止)
   H     H     X    → 短路煞車
```

## 四、完整 GPIO 腳位對照表

| ESP32 GPIO | 方向 | 連接目標 | 說明 |
|:----------:|:----:|:--------|:-----|
| VIN | IN | KIS3R33S #1 → 5V | 主控供電 |
| GND | - | 共同 GND | 必須共地 |
| 3V3 | OUT | TB6612 VCC, TB6612 STBY, Line L/R VCC, HX711 VCC | 3.3V 邏輯供電 |
| **GPIO25** | OUT | TB6612 **PWMA** | 左馬達速度 PWM |
| **GPIO26** | OUT | TB6612 **AIN1** | 左馬達方向 1 |
| **GPIO27** | OUT | TB6612 **AIN2** | 左馬達方向 2 |
| **GPIO32** | OUT | TB6612 **PWMB** | 右馬達速度 PWM |
| **GPIO33** | OUT | TB6612 **BIN1** | 右馬達方向 1 |
| **GPIO14** | OUT | TB6612 **BIN2** | 右馬達方向 2 |
| GPIO17 | IN | KSB042 左側 OUT | 巡線感測器左 |
| GPIO18 | IN | KSB042 右側 OUT | 巡線感測器右 |
| GPIO4 | IN | HX711 DT | 重量資料線 |
| GPIO5 | OUT | HX711 SCK | 重量時脈線 |

## 五、HX711 + Load Cell 接線

```
  ESP32 3V3 ──► HX711 VCC
  ESP32 GND ──► HX711 GND
  HX711 DT  ──► ESP32 GPIO4
  ESP32 GPIO5 ──► HX711 SCK

  HX711 E+ ──► Load Cell 紅線
  HX711 E- ──► Load Cell 黑線
  HX711 A+ ──► Load Cell 白線
  HX711 A- ──► Load Cell 綠線

  Load Cell 固定方式:
  ┌────────────────────────────┐
  │  車體底板                  │
  │    │                      │
  │  [固定端] ── Load Cell ── [活動端]
  │                    │      │
  │              [承重平台]    │  ← 貨物放這裡
  └────────────────────────────┘
```

## 六、ESP32-CAM (DFRobot) 接線

```
  KIS3R33S #2 → 5V ──► ESP32-CAM 5V
  共同 GND        ──► ESP32-CAM GND
  WiFi 連接同一網路，透過 MQTT 通訊

  鏡頭朝下安裝，距地板 10~20cm
  補光 LED (GPIO4) 拍照時短暫點亮

  ⚠️ DFRobot ESP32-CAM 燒錄方式:
     CP2102 TX → CAM GPIO3 (RXD)
     CP2102 RX → CAM GPIO1 (TXD)
     CP2102 GND → GND
     燒錄時: GPIO0 接 GND → 上傳完成 → 拔掉 → 重新上電
```

## 七、電容擺放建議

```
  0.1uF (104) 陶瓷電容 → 放在 IC 電源腳旁邊 (旁路電容)
  位置:
    ① TB6612FNG VCC 腳 與 GND 之間
    ② TB6612FNG VM  腳 與 GND 之間
    ③ ESP32 3V3 腳   與 GND 之間
    ④ HX711 VCC 腳   與 GND 之間

  0.01uF (103) 陶瓷電容 → 更高頻濾波
  位置: 與 0.1uF 並聯在同樣位置

  接法: 電容一腳接 VCC，另一腳接 GND，越靠近 IC 越好

  ❌ 450V 電解電容: 此系統 7.4V，450V 耐壓完全不需要，
                    重量大體積大，不建議使用。
```

## 八、接線 Checklist

- [ ] 所有 GND 共接 (BMS B- / KIS3R33S / TB6612 / ESP32 / CAM / HX711 / Sensor)
- [ ] TB6612 VM 直接接電池 7.4V（不經 KIS3R33S）
- [ ] TB6612 VCC 接 3.3V（不是 5V）
- [ ] TB6612 STBY 接 3.3V（永遠致能）
- [ ] KIS3R33S #1 → ESP32 VIN（5V）
- [ ] KIS3R33S #2 → ESP32-CAM 5V（獨立供電）
- [ ] ESP32-CAM GPIO0 燒錄時接 GND，燒錄完移除
- [ ] 陶瓷電容加在 TB6612 VCC/VM 腳旁
- [ ] Load Cell 固定端鎖緊，活動端可上下移動
- [ ] 巡線感測器距地板 5~10mm，電位器調整靈敏度
